# 🎓 京通 - 多科目备考刷题宝典

[![Platform](https://img.shields.io/badge/Platform-Web%20%7C%20PWA-orange.svg)](#)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](#)

「京通备考刷题宝典」是一款专为公务员、事业单位等公考备考设计的**全栈式多科目智能刷题与复习系统**。系统采用精致的移动端外壳包装（适配桌面与移动端双端体验），提供多账号数据隔离、错题复盘、雷达图数据统计、PWA 离线缓存以及极具特色的趣味刷题模式，助您高效攻克备考难关。

---

## 🌟 核心特性

### 1. 📚 多科目精细化备考
系统内置并支持动态切换多个公考核心科目：
*   **北京·城市市情**：专注地方市情与特色常识考察。
*   **成语·言语700词**：核心言语理解常考成语与高频词汇特训。
*   **时政·月度热点**：紧跟最新时事政治，按月度整理。
*   **政治·理论背诵**：核心政治理论、思想精髓背诵。
*   **理论讲练·挖空**：基于《理论讲练》PDF 抽取——黑体核心表述挖空，穿插考场传真/母题特训选择题（答案对齐书末）。
*   **小黑排坑手册**：备考防坑、易错点与避坑技巧指南。
*   **申论大作文**：名篇范文精读与背诵特训。
*   **申论·分项刷题**：单一题/综合题/公文题/作文题真题精刷（读材料→作答→对照要点→自评）。
*   **技术面试·后端/AI**：简历项目深挖（缓存/短链/Momenta/机器人/小程序/RAG）+ Java/并发/Redis/MySQL/Spring/MQ/网络/AI 基础选择题；另附 `interview_review_handbook.md` 口述问答手册。
*   **资料·速算技巧**：资料分析速算与技巧特训。

### 2. ⚡ 丰富多样的刷题模式
*   **标准模式**：
    *   **顺序练习**：按题库顺序逐题稳步巩固。
    *   **随机测试**：打破常规，随机抽题检验真实水平。
    *   **章节练习**：定位薄弱章节，进行专项突破。
    *   **模拟考试**：随机抽取 20 题，开启限时挑战并生成精美成绩单。
*   **时政/概念特色模式**：
    *   **极速抢答**：每题 10 秒限时，答题越快得分越高，挑战大脑反应极限。
    *   **时间轴排序**：通过拖拽卡片，将时政或历史事件按时间先后进行正确排序。
    *   **关键词配对**：采用直观的 SVG 连线方式，将政策/机构与数字/目标进行配对特训。
    *   **阅读闯关**：阅读政策原文片段，联动攻克 3 道关联深度解析题。
    *   **每日打榜**：每日精选 8 题，完成即可获得今日评级并生成要点速览。

### 3. 📖 申论范文挖空背诵特训
专为申论大作文打造的智能背诵功能，支持多种遮蔽掩膜模式：
*   **全部遮蔽**：整篇挖空，深度默写。
*   **仅挖分论点**：提炼框架，强化文章结构记忆。
*   **仅挖金句**：重点攻克排比句、名言警句及核心表述。
*   **全部展示**：流畅阅读，原文精读对照。

### 4. 🔒 安全隔离与多账号系统
*   支持本地/服务器多账号注册与登录。
*   **错题本与刷题历史隔离**：不同账号的错题记录、收藏题目、学习进度完全独立，互不干扰。
*   **后台管理面板**：管理员可实时查看已注册用户的学习进度，管理用户权限及会员状态。

### 5. 📊 学习进度可视化与系统功能
*   **掌握程度雷达图**：基于 Canvas 动态绘制的章节掌握雷达图，直观展现知识短板。
*   **连续备考打卡**：展示连续学习天数与每日打卡日历，激发备考动力。
*   **错题消灭专项**：错题支持按“全部”与“收藏”分类，并提供“消灭错题”一键重做功能。
*   **考点速查词典**：关键字即时检索，快速查询核心考点。
*   **数据导入导出**：支持导出本地答题记录为 JSON 数据，或导出为 Word 文件备份。

## 🛠️ 技术栈

*   **前端核心**：HTML5, Vanilla CSS (原生 CSS 变量设计，支持深色/浅色主题一键切换)
*   **逻辑控制**：Vanilla JavaScript (ES6+), Web Audio API (音效反馈), Canvas API (粒子背景与图表)
*   **离线缓存**：PWA (Service Worker + Cache API)，支持无网络环境下流畅刷题
*   **后端服务**：Node.js Express 框架，处理多用户注册登录与数据同步

## 🚀 快速上手

### 1. 本地运行 (Node.js 服务)
确保您的计算机上已安装 [Node.js](https://nodejs.org/)。

1. 双击运行目录下的 **`安装开机自启.bat`** 或手动在命令行执行依赖安装：
   ```bash
   npm install
   ```
2. 启动本地服务器：
   *   直接双击 **`启动服务器.bat`**
   *   或者在命令行执行：
       ```bash
       node server.js
       ```
3. 打开浏览器访问：`http://localhost:3000` (或控制台提示的端口号)

*提示：如需在局域网内其他设备访问（如手机），可双击运行 **`开放防火墙端口.bat`**。*

### 2. 静态部署 (Gitee Pages / GitHub Pages)
如果不需要多账号注册登录的服务器端功能，直接作为纯前端项目托管：
1. 将代码推送至 GitHub 或 Gitee 仓库。
2. 开启 **Gitee Pages** 或 **GitHub Pages** 静态托管服务。
3. 访问生成的 `.io` 或 `.gitee.io` 链接即可开始刷题。
